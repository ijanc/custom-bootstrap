# custom bootstrap

This repository aims to centralize the customizations to the
[Bootstrap][3] framework that I use for my projects, *it may not be
useful to you*.

Requirements:

* [Python][0] (webserver)
* [Dart-sass][1] (scss)
* [Pnpm][2] (install [Bootstrap][3])

## How to use

install bootstrap

```
make install
```

watch compilar scss

```
make sass
```

run webserver

```
make serve
```

[0]: https://www.python.org/downloads/
[1]: https://sass-lang.com/install/
[2]: https://pnpm.io/
[3]: https://getbootstrap.com/
